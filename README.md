# Stellar Lift Blog

Static site (Jekyll) hosted on GitHub Pages.

**Live:** https://blog.stellarliftapp.com/

## Sections
- Hydration
- Training
- Recovery

## Structure
- `_posts/` – blog posts (Markdown)
- `_layouts/` – layouts (Jekyll)
- `assets/css/` – styling

## Notes
- Pretty URLs are used (e.g. `/blog/`, `/hydration/`, `/training/`).
- Do not link to `.html` pages; use the folder-style paths.
